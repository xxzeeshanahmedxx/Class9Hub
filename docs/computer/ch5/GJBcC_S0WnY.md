# Creating content with HTML — Class 9 Computer Science Lecture Summary

## Lecture Overview
With the skeleton built, this lecture teaches students how to add actual content to the webpage. The teacher covers the fundamental HTML tags used for text formatting.

## Main Concepts Explained by the Teacher
The instructor demonstrates the Heading tags (`<h1>` to `<h6>`), explaining that `<h1>` is the largest and most important, while `<h6>` is the smallest. Sir then introduces the Paragraph tag (`<p>`) for regular text. The lecture also covers formatting tags like `<b>` for bold and `<i>` for italics to emphasize specific words inside a paragraph.

## Important Definitions and Terms
- **Heading Tags (`<h1>` to `<h6>`):** Used to define HTML headings, automatically making text bold and varying in size.
- **Paragraph Tag (`<p>`):** Used to define a block of text; it automatically adds space before and after the text.

## Key Points and Lists from the Lecture
- Never use `<h1>` just to make text big; it should be used for the main title of the page for SEO purposes.
- You can "nest" tags inside each other. For example: `<p>This is <b>bold</b> text.</p>`.
- The `<br>` tag inserts a single line break and does NOT have a closing tag (it is an "empty" element).

## What Students Should Remember
Listen carefully here: Tag nesting must be done in the correct order! The tag opened last must be closed first. Example: `<b><i>Text</i></b>` is correct. `<b><i>Text</b></i>` is completely wrong and will cause rendering issues.
