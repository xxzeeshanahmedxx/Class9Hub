# Software — Class 9 Computer Science Lecture Summary

## Lecture Overview
This lecture takes a deep dive into Software. While hardware is the body, software is the soul of the computer. The teacher categorizes the different types of software that make a computer usable.

## Main Concepts Explained by the Teacher
The instructor divides software into two main categories: System Software and Application Software. System Software (like Windows or Linux) controls the hardware and provides a platform for other programs. Application Software (like MS Word or games) performs specific tasks for the user. Sir also briefly touches upon Utility Programs like antiviruses.

## Important Definitions and Terms
- **System Software:** Programs designed to run the computer's hardware and application programs.
- **Application Software:** Programs designed for the end-user to perform specific tasks.
- **Operating System (OS):** The most important system software that manages hardware and software resources.

## Key Points and Lists from the Lecture
- **System Software Examples:** Windows, macOS, Android, Device Drivers.
- **Application Software Examples:** Microsoft Office, Chrome, WhatsApp.
- You cannot run Application software without System software installed first.

## What Students Should Remember
Honestly, the distinction between the two is heavily tested. If an exam asks you to classify "Device Drivers", you must put them under System Software, because they directly control hardware components like printers and graphics cards.
