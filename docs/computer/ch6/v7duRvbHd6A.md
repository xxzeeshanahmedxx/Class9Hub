# Cloud storage and data backups — Class 9 Computer Science Lecture Summary

## Lecture Overview
With so much data being generated, this lecture addresses where it is safely stored. The teacher explains Cloud Storage and the absolute necessity of Data Backups to prevent catastrophic loss.

## Main Concepts Explained by the Teacher
The instructor explains that storing data purely on a local hard drive is risky; hardware breaks, gets lost, or stolen. The solution is Cloud Storage—saving data on massive, remote servers accessed via the internet (like Google Drive or OneDrive). The teacher also stresses the 3-2-1 backup rule, ensuring data is copied to multiple locations.

## Important Definitions and Terms
- **Cloud Storage:** A model of computer data storage where digital data is stored on remote servers accessed through the internet.
- **Data Backup:** The process of creating copies of data so that these additional copies can be used to restore the original if it is lost or corrupted.
- **Server Farm:** A massive facility housing thousands of computer servers used for cloud storage.

## Key Points and Lists from the Lecture
- **Pros of Cloud Storage:** Access data from any device anywhere, automatic backups, easy sharing.
- **Cons of Cloud Storage:** Requires an active internet connection, relies on third-party security, ongoing subscription costs.
- Backups defend against hardware failure, ransomware viruses, and human error (accidental deletion).

## What Students Should Remember
Actually, exam questions often ask for the main disadvantage of cloud storage. You must state the "Dependency on Internet Connection." If the Wi-Fi goes down, you completely lose access to all your cloud files, unlike a physical USB drive.
