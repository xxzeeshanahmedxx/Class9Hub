# Network as a System — Class 9 Computer Science Lecture Summary

## Lecture Overview
Chapter 3 shifts the focus from standalone computers to a connected world. This introductory lecture explains the concept of a Computer Network, viewing the entire internet as a massive, interdependent system.

## Main Concepts Explained by the Teacher
The instructor defines a computer network as two or more computers connected together to share resources (like files, printers, or internet access). The teacher recalls the "Theory of Systems" from Chapter 1, explaining that just like a CPU and RAM rely on each other, computers in a network rely on clients, servers, and transmission media to function as a unified whole.

## Important Definitions and Terms
- **Computer Network:** A set of interconnected computers that can share data and resources.
- **Resource Sharing:** The primary goal of a network, allowing multiple users to access the same hardware (like a printer) or software.
- **Node:** Any device (computer, printer, router) connected to a network.

## Key Points and Lists from the Lecture
- Networks save money by allowing an entire office to use just one high-quality printer.
- They allow instantaneous communication (emails, video calls) globally.
- Networks can be wired (Ethernet cables) or wireless (Wi-Fi).

## What Students Should Remember
Listen carefully here: Make sure you remember the primary purpose of a network: "resource sharing". if an exam asks for the biggest advantage of networking in an office, you must explicitly mention that it reduces hardware costs by allowing all employees to share a single peripheral device.
