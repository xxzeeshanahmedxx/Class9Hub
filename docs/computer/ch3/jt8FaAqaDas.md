# Network Protocols — Class 9 Computer Science Lecture Summary

## Lecture Overview
This lecture expands on teh "Rules" of the internet. The teacher explains Network Protocols—the specific languages that computers use to perform different types of tasks over the web.

## Main Concepts Explained by the Teacher
The instructor clarifies that you cannot use the same rules to send an email as you do to load a webpage. Therefore, different protocols exist. The teacher introduces HTTP (for loading websites), FTP (for downloading/uploading files), and SMTP (for sending emails). The lecture also covers TCP/IP, the foundational suite that makes the entire internet function.

## Important Definitions and Terms
- **TCP/IP (Transmission Control Protocol / Internet Protocol):** The core suite of communication protocols used to interconnect network devices on the internet.
- **HTTP (Hypertext Transfer Protocol):** The protocol used for transmitting web pages.
- **SMTP (Simple Mail Transfer Protocol):** The protocol used for email transmission.

## Key Points and Lists from the Lecture
- **FTP (File Transfer Protocol):** Used to securely transfer large files between a client and a server.
- TCP breaks data into small "packets" and ensures they all arrive safely.
- IP handles the actual addressing and routing of those packets.

## What Students Should Remember
Well, matching the acronym to its function is critical for exams. If a question asks "Which protocol is used by web browsers to display websites?", you must select HTTP. If it asks about "uploading a document to a server," select FTP.
