# Network security — Class 9 Computer Science Lecture Summary

## Lecture Overview
With computers connected globally, this lecture covers Network Security. The teacher explains the threats that exist on the internet and the methods used to protect data as it travels across vulnerable cables and Wi-Fi waves.

## Main Concepts Explained by the Teacher
Sir introduces the concept of Cyber Security. He explains that hackers can intercept data while it is in transit. To prevent this, data is secured using Encryption—scrambling the message so that only the authorized receiver with a specific "key" can read it. The lecture also reviews Firewalls as a barrier against unauthorized network access.

## Important Definitions and Terms
- **Network Security:** Policies and practices adopted to prevent unauthorized access, misuse, or modification of a computer network.
- **Encryption:** The process of converting plain, readable text into an unreadable cipher to secure it from hackers.
- **Decryption:** The process of converting the encrypted cipher back into readable text using a key.

## Key Points and Lists from the Lecture
- Passwords and User IDs are the first line of defense (Authentication).
- Firewalls block suspicious incoming network traffic.
- Encryption ensures that even if data is stolen during transmission, it remains unreadable and useless to the thief.

## What Students Should Remember
Honestly, the definition of Encryption is a guaranteed short question. You must use the terms "plain text" and "cipher text". Stating that encryption "hides data by scrambling it into cipher text" will secure you full marks.
