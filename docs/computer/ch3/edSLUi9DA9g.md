# Network Topologies — Class 9 Computer Science Lecture Summary

## Lecture Overview
This highly visual lecture covers Network Topologies. The teacher explains teh physical and logical layout of how computers are wired together in a network.

## Main Concepts Explained by the Teacher
The instructor draws the different wiring layouts. In a Bus topology, all computers connect to a single central cable (the backbone). In a Star topology, all computers connect to a central Hub or Switch. In a Ring topology, computers are connected in a closed loop. Finally, in a Mesh topology, every computer is connected directly to every other computer. The teacher highlights the pros and cons of each layout.

## Important Definitions and Terms
- **Topology:** The geometric arrangement of a computer network.
- **Bus Topology:** All devices share a single communication cable.
- **Star Topology:** All devices connect to a central node (Switch/Hub).

## Key Points and Lists from the Lecture
- **Bus:** Cheap, but if the main cable breaks, the whole network fails.
- **Star:** Most common today. If one cable breaks, only one PC goes offline. If the central Switch breaks, the whole network dies.
- **Mesh:** Extremely reliable (many backup paths) but very expensive and difficult to wire.

## What Students Should Remember
So basically, the Star Topology is the most important one to memorize because it is what modern schools and homes use. You must be able to draw its diagram—a central box (Switch) with lines radiating outward to multiple computers.
