# Security and maintenance — Class 9 Computer Science Lecture Summary

## Lecture Overview
This final lecture for Chapter 2 shifts from hardware to software. The teacher covers Security and Maintenance, focusing on protecting the computer from digital threats and keeping the operating system running efficiently.

## Main Concepts Explained by the Teacher
The instructor explains that digital maintenance is just as important as physical cleaning. Sir introduces tools like Disk Defragmenter (which organizes scattered files to speed up the hard drive) and Disk Cleanup (which deletes temporary junk files). The lecture then covers security: the absolute necessity of installing Antivirus software to block malware, and using a Firewall to prevent hackers from accessing the network.

## Important Definitions and Terms
- **Disk Defragmentation:** A utility that rearranges fragmented data on a mechanical hard drive so files are stored contiguously, improving read speeds.
- **Antivirus:** Software designed to detect, block, and remove malicious programs (malware, viruses, trojans).
- **Firewall:** A network security system that monitors and controls incoming and outgoing network traffic.

## Key Points and Lists from the Lecture
- A slow, freezing computer might not be broken hardware; it might be heavily fragmented or infected.
- Always keep the OS and antivirus definitions updated to protect against new threats.
- Regular data backups are the ultimate defense against ransomware or hard drive failure.

## What Students Should Remember
To be fair, examiners love the difference between Antivirus and a Firewall. Remember: Antivirus acts like a doctor finding and killing sickness *inside* the computer. A Firewall acts like a security guard standing at the *door* (network connection), preventing bad traffic from entering in the first place.
