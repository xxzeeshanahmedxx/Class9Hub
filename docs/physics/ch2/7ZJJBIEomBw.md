# Resultant Vector — Class 9 Physics Lecture Summary

## Lecture Overview
This session introduces the Resultant Vector. The teacher explains how to mathematically and graphically combine multiple vectors into a single vector that has the exact same physical effect.

## Main Concepts Explained by the Teacher
When adding scalars, we use simple arithmetic (2+2=4). But for vectors, direction matters. The teacher explains the Head-to-Tail rule for adding vectors graphically. The resultant vector is drawn from the tail of the first vector to the head of the last vector.

## Important Definitions and Terms
- **Resultant Vector:** A single vector that has the combined effect of all the individual vectors added together.
- **Head-to-Tail Rule:** A graphical method of adding vectors by placing the tail of the second vector on the head of the first.

## Key Points and Lists from the Lecture
- Vector addition is not simple arithmetic; geometry is required.
- You can add any number of vectors using the Head-to-Tail rule.
- The order of addition does not matter ($\vec{A} + \vec{B} = \vec{B} + \vec{A}$).

## What Students Should Remember
Well, Make sure you practice drawing the Head-to-Tail rule using a ruler and protractor. The direction of the resultant vector is always from the start point to the final end point.
