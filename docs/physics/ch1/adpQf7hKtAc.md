# Significant Figures — Class 9 Physics Lecture Summary

## Lecture Overview
This lesson introduces Significant Figures, a foundational concept for expressing the precision of a measurement. The teacher explains that writing down numbers randomly in physics is incorrect; we must only write the digits we are mathematically sure about. This ensures laboratory calculations maintain scientific integrity.

## Main Concepts Explained by the Teacher
The teacher defines significant figures as all the accurately known digits plus the first doubtful digit in a measurement. The lecture is heavily focused on the mathematical rules used to determine which digits in a number are significant, particularly how to treat the number zero depending on its placement.

## Important Definitions and Terms
- **Significant Figures:** The reliably known digits in a measurement along with the first uncertain digit.
- **Doubtful Digit:** The last digit in a measured value, which contains the instrument's uncertainty.

## Key Points and Lists from the Lecture
**Rules for Significant Figures:**
1. All non-zero digits (1-9) are always significant.
2. Zeros located between two significant figures are significant (e.g., 205 has 3).
3. Leading zeros (zeros to the left) are strictly for spacing the decimal and are NEVER significant (e.g., 0.02 has 1).
4. Trailing zeros (zeros to the right) in a decimal fraction ARE significant (e.g., 3.500 has 4).

## What Students Should Remember
Honestly, The most heavily tested rule in 9th class exams is the treatment of zeros. You guys must memorize that leading zeros never count, while trapped and trailing zeros (in decimals) always count.
