# States of Equilibrium — Class 9 Physics Lecture Summary

## Lecture Overview
This lesson explores the three States of Equilibrium: Stable, Unstable, and Neutral. The teacher explains how an object's Centre of Gravity (CG) determines how it reacts when slightly disturbed.

## Main Concepts Explained by the Teacher
The teacher uses visual props. **Stable Equilibrium:** If tilted, the object's CG rises, and gravity pulls it back to its original position (e.g., a cone resting on its broad base). **Unstable Equilibrium:** If tilted, the CG lowers, and the object topples over completely (e.g., a pencil balanced on its sharp tip). **Neutral Equilibrium:** If moved, the CG stays at the same height, and the object simply stays in its new position (e.g., a rolling ball).

## Important Definitions and Terms
- **Stable Equilibrium:** A state where an object returns to its original position after a slight tilt.
- **Unstable Equilibrium:** A state where an object topples over after a slight tilt.
- **Neutral Equilibrium:** A state where an object remains in its new displaced position.

## Key Points and Lists from the Lecture
- The position of the Centre of Gravity dictates stability.
- Stable $\rightarrow$ CG rises when tilted.
- Unstable $\rightarrow$ CG lowers when tilted.
- Neutral $\rightarrow$ CG remains horizontal.

## What Students Should Remember
Honestly, The behavior of the CG is heavily tested in subjective questions. Make sure you be able to draw simple diagrams showing how the vertical line of weight shifts outside the object's base to cause toppling in unstable equilibrium.
