# Rules of Binomial Nomenclature — Class 9 Biology Lecture Summary

## Lecture Overview
Following the concept of naming, this lecture details the strict typographical Rules of Binomial Nomenclature. The teacher explains exactly how to write these Latin names properly in exams to avoid losing marks.

## Main Concepts Explained by the Teacher
The instructor outlines three absolute rules. First, the Genus name must ALWAYS start with a capital letter, while the species name must be entirely lowercase. Second, when printed in a book, the name must be *italicized*. Third, when handwritten by a student, the Genus and Species names must be underlined separately.

## Important Definitions and Terms
- **Typographical Formatting:** The strict rules regarding capitalization and italics/underlining when writing scientific names.
- **Abbreviation:** After using a full name once, the Genus can be abbreviated (e.g., *Escherichia coli* becomes *E. coli*).

## Key Points and Lists from the Lecture
- **Correct format:** *Homo sapiens* (italicized in print).
- **Correct handwritten format:** <u>Homo</u> <u>sapiens</u>.
- **Common scientific names to memorize:** Onion (*Allium cepa*), Common House Crow (*Corvus splendens*), Starfish (*Asterias rubens*).

## What Students Should Remember
Pro tip: Writing the name correctly in your board exam is critical. If you write "Homo Sapiens" (with a capital S) or fail to underline it when writing by hand, the examiner will mark it completely wrong due to a syntax error. Capitalize the first letter, lowercase the second word, and underline both separately!
