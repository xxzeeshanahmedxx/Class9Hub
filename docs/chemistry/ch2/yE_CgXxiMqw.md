# Isotopes And Their Masses — Class 9 Chemistry Lecture Summary

## Lecture Overview
This lesson introduces Isotopes, a concept that explains why all atoms of the same element are not strictly identical in mass. The teacher clarifies how varying neutron counts create different versions of the same element.

## Main Concepts Explained by the Teacher
Sir defines Isotopes as atoms of the same element that have the exact same Atomic Number (protons) but different Mass Numbers (neutrons). The teacher uses Hydrogen as a prime example, detailing its three isotopes: Protium (0 neutrons), Deuterium (1 neutron), and Tritium (2 neutrons). 

## Important Definitions and Terms
- **Isotopes:** Atoms of an element having the same atomic number but different mass numbers.
- **Protium:** The most common isotope of hydrogen, containing 1 proton and 0 neutrons.
- **Deuterium:** Heavy hydrogen, containing 1 proton and 1 neutron.
- **Tritium:** Radioactive hydrogen, containing 1 proton and 2 neutrons.

## Key Points and Lists from the Lecture
- Isotopes have identical chemical properties because chemical reactions depend only on the number of electrons.
- Isotopes have different physical properties (like mass and density) due to the different number of neutrons.
- Carbon has three isotopes: C-12, C-13, and radioactive C-14.

## What Students Should Remember
Look, the fact that isotopes have *identical chemical properties* but *different physical properties* is a guaranteed exam question. Because they have the same number of electrons in their outer shells, they react exactly the same way in chemical equations.
