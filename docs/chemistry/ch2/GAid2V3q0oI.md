# Atomic Number and Mass Number — Class 9 Chemistry Lecture Summary

## Lecture Overview
With the subatomic particles identified, this lecture explains how we mathematically define and identify different elements using their Atomic Number and Mass Number.

## Main Concepts Explained by the Teacher
Sir defines the Atomic Number ($Z$) as the total number of protons in the nucleus of an atom. This number is the strict "identity card" of an element. The teacher then defines the Mass Number ($A$) as the sum of protons and neutrons in the nucleus. The lecture shows how to calculate the number of neutrons by subtracting $Z$ from $A$.

## Important Definitions and Terms
- **Atomic Number ($Z$):** The number of protons present in the nucleus of an atom.
- **Mass Number ($A$):** The total number of protons and neutrons in an atom's nucleus.
- **Nucleons:** A collective term for protons and neutrons (the particles inside the nucleus).

## Key Points and Lists from the Lecture
- Every atom of Carbon has exactly 6 protons ($Z=6$). If it had 7, it would be Nitrogen.
- In a neutral atom, the number of electrons exactly equals the number of protons.
- Formula to find neutrons: $n = A - Z$.
- Symbolic representation: $_{Z}^{A}\text{X}$ (where X is the element symbol).

## What Students Should Remember
Well, do not confuse the two numbers! The Atomic Number ($Z$) is usually the smaller number and defines the element. The Mass Number ($A$) is the larger number. Knowing the formula $n = A - Z$ is essential for solving numericals in this chapter.
