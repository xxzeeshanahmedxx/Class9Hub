# Coordinate Covalent Bond — Class 9 Chemistry Lecture Summary

## Lecture Overview
This video introduces the third type of bond: the Coordinate Covalent Bond (or Dative Bond). The teacher explains a special case of sharing where one atom acts incredibly generously.

## Main Concepts Explained by the Teacher
In a normal covalent bond, each atom contributes one electron to make a pair. The teacher explains that in a Coordinate Covalent Bond, *both* electrons in the shared pair are provided by just ONE atom (the Donor). The other atom (the Acceptor) provides an empty orbital. The classic example used is the formation of the Ammonium ion ($NH_4^+$) from Ammonia ($NH_3$) and a Hydrogen ion ($H^+$).

## Important Definitions and Terms
- **Coordinate Covalent Bond (Dative Bond):** A type of covalent bond where the shared electron pair is donated by only one of the bonded atoms.
- **Donor:** The atom that provides the electron pair (it must have a lone pair).
- **Acceptor:** The atom that accepts the electron pair (it must be deficient in electrons).

## Key Points and Lists from the Lecture
- It is represented by an arrow ($\rightarrow$) pointing from the donor to the acceptor, instead of a simple straight line ($-$).
- Ammonia ($NH_3$) has a lone pair on Nitrogen, making it an excellent donor.
- A hydrogen ion ($H^+$) has zero electrons, making it an excellent acceptor.

## What Students Should Remember
Look, drawing the arrow correctly is everything in a dative bond. If you draw a normal straight line, it implies a standard covalent bond. The arrow MUST point from the atom with the lone pair (Donor) to the atom needing the electrons (Acceptor).
